Status Quo

IRC

  • Enables quick discussion and collaboration without breaking workflow.
  • Logs of discussion can be valuable for future references
  • Open, Widely Supported Protocol
  • Easily extensible with other tools
  • Easy way to communicate with the OSS communities spread around the world

Our experience with IRC

  • low internet bandwidth
  • frequent disconnection of internet
  • sometimes no connection at all
  • IRC ports blocked on college network
  • No persistent logs with existing web IRC clients

IRC has it's learning curve

  • IRC servers(IPs, ports, SSL)
  • channels
  • PM
  • Commands

We were sad :(

  • Centralized logging
  • 24x7 idling
  • Mail notifications when away
  • Search interface for chat logs
  • Realtime
  • HTML5 mobile interface for Waartaa
  • Easy setup and deployment
